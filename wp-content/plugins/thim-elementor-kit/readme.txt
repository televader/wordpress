Thim Elementor Kit is plugin support build theme, layout, page, post, product, woocommerce, learnpress, courses with Elementor.

== Description ==

If you are a fan of Elementor Page Builder and always want to have your own header and footer but for free.
Thim-Kit is your completely perfect solution with powerful features that make you love this plugin immediately.
What makes Thim-Kit special is the flexiblity in customizing header, footer and some other blocks in one page or all pages.
Also, you can create Mega Menu, edit product layout, edit blog layout by your Basic Elementor.

Especially, for Eduma or other LMS WordPress Themes users, Thim-Kit has ThimPress Extension Widgets that allow you to edit and customize courses.
Each icon, each title, and many other things now will be made following your own style.
We focus on the asy-to-use, that is why drag & Drop Builder is also compatible with Thim-Kit and you can choose what you want from our already built templates, widgers.
Powerful as Elementor but free, Thim-Kit is the best choice for your website, especially LMS website.

== Installation ==

**From your WordPress dashboard**
1. Visit 'Plugin > Add new'.
2. Search for `thim elementor kit`.
3. Activate LearnPress from your Plugins page.

**From WordPress.org**
1. Search, select and download `thim-elementor-kit`.
2. Upload the `thim-elementor-kit` folder to the `/wp-content/plugins/` directory or alternatively upload the thim-elementor-kit.zip file via the plugin page of WordPress by clicking 'Add New' and selecting the zip from your computer.
